Here’s a practical way to map your **Venue Deal Sheet** + **Rehearsal Block Tracker** into **WildApricot Event entry fields**, so each booking/project lives as an “Event” with consistent info.

## Recommended WildApricot setup

### 1) Create 3 Event templates (or categories)
- **Rehearsal Block (Studio Rental)**
- **Performance (Theater/Bar)**
- **Video Shoot**

### 2) Add these **Custom Event Fields** (copy/paste names)

**Core**
- Venue Name (text)
- Venue Address (text)
- Venue Contact Name (text)
- Venue Contact Email/Phone (text)
- CAN.dy Lead Co-Director (dropdown/text)
- Day-of Contact (text)

**Schedule**
- Load-in Time (time)
- Start Time (time)
- End Time (time)
- Strike/Exit Time (time)
- Notes on Access/Parking (text area)

**Rules / Technical**
- Capacity / Occupancy (number)
- Heels Allowed? (yes/no)
- Chair Use Allowed? (yes/no)
- Sound Available? (yes/no)
- Sound Input (dropdown: Aux/Bluetooth/XLR/Other)
- Recording Permitted? (yes/no)
- Recording Permission Link (URL) *(paste email/pdf link in Drive)*

**Money**
- Base Fee (currency)
- Deposit Amount (currency)
- Deposit Due Date (date)
- Additional Fees (currency)
- Additional Fees Detail (text area)
- Cancellation/Reschedule Terms (text area)
- Payment Method (dropdown: card/ACH/check/other)
- Payment Deadline (date)

**Insurance / Contract**
- Insurance Required? (yes/no)
- Insurance Amount Required (currency/text)
- Additional Insured Required? (yes/no)
- Contract Link (URL) *(Drive link)*
- Contracting Party Listed Correctly? (yes/no)

**Approvals (bylaws compliance)**
- Co-Director Approval #1 (yes/no)
- Co-Director Approval #2 (yes/no)
- Other Co-Director Reviewed (yes/no)
- Treasurer Notified of Deadlines (yes/no)
- Secretary Saved Docs (yes/no)

**Ticketing / Revenue (performance only)**
- Tickets Sold By (dropdown: CAN.dy/Venue/Other)
- Ticketing Link (URL)
- Door Split / Guarantee / Bar Minimum (text area)
- Merch Allowed? (yes/no)
- Age Policy Notes (text area)

**Budget sanity**
- Expected Income (currency)
- Expected Expenses (currency)
- Deficit Approved in Writing? (yes/no)

### 3) Attachments / links
WildApricot events don’t always handle attachments elegantly, so best practice is:
- Keep a folder per Event in Google Drive, and paste links into:
  - **Contract Link**
  - **Recording Permission Link**
  - (optional) **Invoice/Quote Link**
